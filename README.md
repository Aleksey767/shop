# Launch Guide
1.npm install  
2.npm run build/dev  
3.npm run start/node  
4.Открыть браузер и ввести http://localhost:3000  
# Admin-mode Guide  
1.Добавить в конце строки http://localhost:3000 - /admin  
2.Ввести данные:   
Логин: admin   
Пароль: wgforge    
3.Чтобы выйти из режима нужна нажать Logout в правом верхнем углу
